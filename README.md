# Flask Authentication System

A secure authentication system built with Flask and PostgreSQL. It allows users to register, log in, reset passwords using OTP, and manage their accounts securely.

## 🚀 Features

- User Registration
- User Login & Logout
- Forgot Password with OTP Verification
- Reset Password
- Change Password
- Password Hashing using Bcrypt
- Profile Management
- Session-Based Authentication
- PostgreSQL Database Integration

## 🛠️ Tech Stack

- Python
- Flask
- Flask-Login
- Flask-WTF
- Flask-Mail
- Flask-Bcrypt
- SQLAlchemy
- PostgreSQL
- HTML
- CSS
- Jinja2

## ⚙️ Installation

```bash
git clone https://github.com/ramgupta639225-svg/Flask-Authentication-System.git

cd Flask-Authentication-System

pip install -r requirements.txt

python app.py
```
